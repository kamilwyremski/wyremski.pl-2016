The DEMOTY2 script allows you to create your own website with demotivators and memes in an easy and fast way. It allows generating demotivators and memes with possibility to edit their later. It is a modern script written in accordance with modern standards based on the Twig2 template system and the CSS framework Bulma.

The script has been optimized so you can install it on almost any hosting and will work properly with a large number of images and users.
